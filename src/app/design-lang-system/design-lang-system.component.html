Design Lab Design Lab I experiment and play with different components and prototype new features. Particularly, I define a Design Language System. It is a way to define design guidelines that can be reused. Spec are assigned to each web component instead of a whole page. In such way, when these components are used in different pages, design can be seamlessly applied through the whole website. It is a modularized way to design, and aligns with the Reuse Spirit in software development.
